# Watermark 组件规范（vc-design）

基于 Ant Design 5 的 Watermark（水印），用于在页面或容器上添加文字/图片水印，通过配置旋转角度、间距与字体样式统一品牌展示与防泄露效果。

---

## 1. 用途

- **用途**：在页面或某个容器区域内叠加重复水印，常用于展示环境信息（如「内部测试」）、账号标识或品牌文案。
- **使用方式**：`<Watermark content=\"内部测试\">...</Watermark>` 包裹需要打水印的区域。

---

## 2. 文本与图片

| 属性 | 说明 |
|------|------|
| **content** | 文本内容，字符串或字符串数组（数组时多行显示） |
| **image** | 图片地址，与 `content` 互斥，优先使用图片 |
| **font** | 文本样式配置，包括 `color`、`fontSize`、`fontWeight`、`fontStyle`、`fontFamily`、`textAlign` |

- 文本颜色、字号建议与品牌规范保持一致，可结合 `vcTokens` 中的文本色和字号 token 进行设计（在主题中统一配置）。\n+

---

## 3. 尺寸、间距与偏移

| 属性 | 说明 |
|------|------|
| **width** / **height** | 单个水印的宽高（数字），用于控制绘制区域大小 |
| **gap** | 水印间的水平/垂直间距，`[gapX, gapY]` |
| **offset** | 整体偏移量，`[offsetLeft, offsetTop]`，用于调整起始绘制位置 |
| **rotate** | 旋转角度，单位度，常用如 -22° |

- 间距与尺寸应根据内容密度和可读性调整，避免过密或过稀。

---

## 4. 容器与继承

| 属性 | 说明 |
|------|------|
| **children** | 需要添加水印的子内容（容器） |
| **inherit** | 是否继承父节点的尺寸与滚动区域，默认为 true（跟随所在容器） |
| **zIndex** | 水印层级，默认高于内容但低于弹窗/浮层，可根据需要调整 |

---

## 5. 样式与回调

| 属性 | 说明 |
|------|------|
| **style** / **className** / **rootClassName** | 控制 Watermark 容器样式与类名 |
| **onRemove** | 水印被移除时回调（如卸载或条件变更时） |

---

## 6. 设计建议

- 文本水印应半透明、字号适中，避免影响主要内容阅读；可结合 `rgba` 或半透明品牌色。
- 重要页面（如预览、导出页）可增加水印密度或使用更明显的文案（如「仅供内部使用」）。+
